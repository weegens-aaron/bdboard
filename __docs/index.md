# bdboard Documentation (maintainer)

FlowDoc-style maintainer documentation for **bdboard** — a Python + FastAPI +
HTMX dashboard for `bd` (beads) workspaces. These docs describe what the code
**does** (behavior, flows, contracts), not a file inventory.

## Start here

- **[Architecture](Architecture.md)** — the system overview: tech stack,
  system diagram, features, flows, dependencies, directory guide, API surface,
  and views.
- **[_FlowDocGuide](_FlowDocGuide.md)** — the authoring contract (templates +
  rules) every doc follows.
- **[_Manifest](_Manifest.md)** — progress tracker for the full doc set.

## Sections

- **[Features](Features/index.md)** — user-facing capabilities, end to end.
- **[Flows](Flows/index.md)** — multi-step journeys through the system.
- **[Endpoints](Endpoints/index.md)** — every HTTP route, request to response.
- **[Views](Views/index.md)** — the three server-rendered pages.
- **[Concepts](Concepts/index.md)** — cross-cutting design ideas.
