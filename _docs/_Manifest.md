# FlowDoc Manifest (user)

> Single source of truth for what gets documented in the **user edition**. Each
> item below has exactly one doc bead (parented to the discovery bead
> `bdboard-mol-oot`). When a doc is written, tick its box `- [ ]` -> `- [x]`
> and bump the progress counters.
>
> IDs are **banded / type-segmented** by Diataxis group (Getting-Started 001+,
> Guides 010+, Tutorials 040+, Reference 060+, Concepts 090+) so the group is
> readable from the number and new items slot in without renumbering. Section
> dirs and file names are **PascalCase**; the link target must equal the
> filename exactly.
>
> Delivery model: **installed-locally** (you install and run bdboard on your
> own machine), so the Getting-Started band includes an Installation page.
