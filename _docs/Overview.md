# bdboard

> Your project's work, shown as a live board in your browser.

## What Is It

bdboard is a small tool you run on your own computer. When you start it, it opens
a page in your web browser that shows everything your team is tracking as a set of
cards arranged into lanes — what's ready to pick up, what's in progress, what's
blocked, and what was recently finished. The board refreshes itself the moment
anything changes, so you never have to reload.

Alongside the board you also get a **History** page for looking back over time, a
**Memory** page for your project's saved notes, and a **Formulas** feature for
spinning up a whole batch of related work items from a template.

## Who Is It For

bdboard is for anyone who works against a project's issue tracker and wants to
*see the shape of the work* instead of running one query after another:

- **People picking up work** who want to know "what can I start right now?"
- **People reviewing progress** who want a glance-able view of what's moving.
- **People looking back** who want to know what got finished last month and how
  long things take.

You don't need to be the person who set the project up. If someone hands you the
project folder, you can run bdboard against it.

## Key Features

- **A live board** that buckets every item into Epics plus the lanes
  *Deferred / Blocked / Ready / In Progress / Closed*, newest and
  highest-priority first.
- **Bead details** — click any card to read everything about an item, see its
  timeline, and (for open items) fix a field right in the page.
- **History & trends** — a created-vs-closed chart and a searchable list of
  finished work over weeks or months.
- **Memory** — browse, search, add, and remove your project's saved notes.
- **Formulas** — pour a template to create a structured batch of related items
  in one go.
- **Always current** — every page updates itself live as the project changes,
  in every open tab.

## Requirements

- A computer where you can install and run a command-line tool.
- The project folder you want to look at (the one your team tracks work in).
- A modern web browser (bdboard opens one for you).
