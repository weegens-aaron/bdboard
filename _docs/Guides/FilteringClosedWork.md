# How to filter recently closed work

## What You'll Learn

How to control which finished work shows on the board, the difference between the
board's short window and the History screen's long view, and when to use each.

## Prerequisites

- bdboard running with your project open — see the
  [Quick Start](../Getting-Started/QuickStart.md).

## Overview

The board keeps a deliberately short memory of finished work so it stays focused
on what's current. You can adjust that window with the board's time filter, and
when you need to look further back, you switch to the
[History screen](../Reference/HistoryScreen.md).

## Step 1: Find the board time filter

At the top of the board, above the lanes, is a small set of range buttons:
**12h**, **1d**, and **3d**. The default is **1d**.

**Expected result:** you can see the three range buttons, with one highlighted as
active.

## Step 2: Choose a window

Click a range to change how far back the **Closed** lane reaches.

**Expected result:** the Closed lane shows only work finished within the window
you picked, and the closed count in the top bar matches it.

> [!NOTE]
> The board's longest window is 3 days on purpose. It's a *recent* view — for
> anything older, use the History screen.

## Step 3: Read the Closed lane

The **Closed** lane lists items finished inside the active window, newest-first.

**Expected result:** the most recently finished work sits at the top of the lane.

## Step 4: Go further back with History

When you need weeks or months instead of days, click **History** in the top bar.
There you get **7d / 30d / 90d / All** ranges plus a custom date picker.

**Expected result:** the History screen opens with a much longer reach over
finished work. See [Review history and trends](ReviewingHistory.md).

> [!TIP]
> Rule of thumb: use the **board** filter for "what did we just finish?" and the
> **History** screen for "what got done last week/month?".

## Troubleshooting

| Problem | Fix |
|---------|-----|
| The Closed lane is empty | Widen the board window, or check the [History screen](../Reference/HistoryScreen.md) for older work. |
| I can't find something I closed last week | The board only reaches 3 days; use History with a wider range. |
| The closed count doesn't match the lane | Both follow the same window; switch the filter once to resync. |
